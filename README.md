# .dotfiles
dotfile configs for my favourite tools


## Instructions
Get this repo by running
```
cd
git clone https://github.com/Eloquencere/.dotfiles.git
```

Run the script in .bin/
```
cd ~/.dotfiles/.bin/
source software_install_1.sh
```

After Restart run
```
cd ~/.dotfiles/.bin/
source software_install_2.sh
```
